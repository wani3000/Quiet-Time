말씀묵상 앱

1. 개요

말씀묵상 앱(이하 "앱")은 사용자의 개인정보를 매우 중요하게 생각하며, 개인정보 보호법 등 관련 법령을 준수하고 있습니다.

핵심 원칙: 말씀묵상 앱은 사용자의 개인정보를 수집하지 않으며, 모든 데이터는 사용자의 기기에만 저장됩니다.

2. 수집하지 않는 정보

말씀묵상 앱은 다음과 같은 정보를 일체 수집하지 않습니다:

  • 📧 이메일 주소
  • 📱 전화번호
  • 🆔 사용자 계정 정보
  • 📍 위치 정보
  • 📞 연락처 정보
  • 🖼️ 사진 및 미디어 (저장 목적 외)
  • 💳 결제 정보
  • 📊 사용 통계 또는 분석 데이터

3. 기기 권한 사용

앱의 정상적인 작동을 위해 다음 권한을 요청합니다:

권한 목적 필수 여부
📸 사진 라이브러리 (저장) 사용자가 생성한 말씀 카드를 기기에 저장하기 위함 선택 (기능 사용 시 필요)

3.1 사진 라이브러리 권한 상세

  • 사용 시점: 사용자가 말씀 카드를 저장하려고 할 때만
  • 사용 범위: 말씀 카드 이미지를 사진 보관함에 저장
  • 중요: 기존 사진을 읽거나 업로드하지 않습니다
  • 거부 시: 말씀 카드 저장 기능만 사용할 수 없으며, 다른 모든 기능은 정상 작동

4. 로컬 저장 데이터

다음 정보는 사용자의 기기에만 저장되며, 외부로 전송되지 않습니다:

데이터 종류 저장 위치 목적
묵상 메모 기기 내부 저장소 (SharedPreferences) 사용자의 개인 묵상 기록 보관
앱 설정 기기 내부 저장소 사용자 선호 설정 유지
방문 날짜 기록 기기 내부 저장소 읽은 말씀 표시

중요: 앱을 삭제하면 위의 모든 데이터가 함께 삭제됩니다. 복구할 수 없으므로 중요한 묵상 메모는 따로 백업하시기 바랍니다.

5. 제3자 서비스

말씀묵상 앱은 다음과 같은 제3자 서비스를 사용하지 않습니다:

  • ❌ 광고 네트워크
  • ❌ 분석 도구 (Google Analytics, Firebase Analytics 등)
  • ❌ 소셜 미디어 SDK
  • ❌ 크래시 리포팅 도구
  • ❌ 클라우드 저장소

따라서 제3자에게 사용자 정보가 공유되지 않습니다.

6. 공유 기능

앱 내 공유 기능 사용 시:

  • 📤 iOS 기본 공유 시트를 사용합니다
  • 🎯 사용자가 선택한 앱(메시지, 카카오톡, 인스타그램 등)으로만 공유됩니다
  • 💾 공유된 말씀 카드 이미지는 일시적으로 생성되며, 앱에 저장되지 않습니다
  • 🔒 공유 과정에서 개발자는 어떤 정보도 수집하지 않습니다

7. 데이터 보안

말씀묵상 앱은 사용자 데이터 보호를 위해 다음과 같은 조치를 취합니다:

  • 🔐 모든 데이터는 iOS 샌드박스 내에서만 저장됩니다
  • 🚫 네트워크 통신이 없어 외부 침입 위험이 없습니다
  • 📱 iOS 보안 정책에 따라 다른 앱이 데이터에 접근할 수 없습니다
  • 🔄 정기적인 보안 업데이트를 제공합니다

8. 아동 개인정보 보호

말씀묵상 앱은 모든 연령대의 사용자를 대상으로 하며, 아동(만 14세 미만)의 개인정보를 수집하지 않습니다.

아동이 본 앱을 사용하더라도 개인정보가 수집되지 않으므로, 부모의 동의 없이 안전하게 사용할 수 있습니다.

9. 개인정보 보호정책 변경

본 개인정보 보호정책은 관련 법령 및 앱 기능 변경에 따라 업데이트될 수 있습니다. 정책이 변경되는 경우:

  • 📢 앱 업데이트 노트에 변경 사항을 공지합니다
  • 📅 변경된 정책은 공지 후 7일 뒤부터 시행됩니다
  • 🔗 최신 정책은 항상 이 페이지에서 확인할 수 있습니다

10. 사용자의 권리

말씀묵상 앱은 개인정보를 수집하지 않지만, 사용자는 다음과 같은 권리를 가집니다:

  • ✅ 로컬 저장된 데이터 확인 (앱 내에서)
  • 🗑️ 저장된 묵상 메모 삭제 (앱 내에서)
  • 📴 권한 설정 변경 (iOS 설정 → 말씀묵상)
  • 🔄 앱 삭제를 통한 모든 데이터 완전 제거
